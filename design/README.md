# design/

Design explorations for pendeln, made with Claude Design and exported as a handoff bundle.

- `handoff/Pendeln Explorations.dc.html` — the exploration canvas (screens as HTML/CSS prototypes). Open it in a browser to see the current visual direction: transit-system clarity, warm off-white/ink, one signal accent, stations-on-a-line progress.
- `handoff/uploads/` — reference material fed into the design session.

These are prototypes, not production UI. The build phase recreates them properly (React Native / Expo) — see [docs/06-handoff-notes.md](../docs/06-handoff-notes.md) for the rules a design or coding session has to keep.
